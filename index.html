Luawie 3

Wii controllers on your pc. Simulate keyboard and mouse inputs using a simple Lua configuration file 

Keyboard.Up     = Guitar.joystick.Up
Keyboard.Down   = Guitar.joystick.Down
Keyboard.Return = Guitar.StrumUp or Guitar.StrumDown
Keyboard.Escape = Guitar.Minus or Guitar.Plus

It's that easy!
